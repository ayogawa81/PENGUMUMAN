📱 Cara Praktis Update Jadwal via HP

Sistem Mandiri & Stabil Aktif!
Mengubah teks langsung dari repositori GitHub HP adalah cara paling aman untuk menghindari crash pemblokiran browser.

1. Buka aplikasi GitHub di HP.

2. Masuk ke repositori PENGUMUMAN -> file index.html.

3. Klik tombol Edit (Ikon Pensil).

4. Cukup ganti teks Jam, Kegiatan, atau Ruangan pada baris jadwal di atas tanpa merusak tanda kurung siku <...>.

5. Klik **Commit Changes** (Simpan).
